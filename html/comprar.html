Como Comprar?

  1. Iniciar la Compra:

    2. Se ejecuta la función comprarproducto(_) para comenzar el proceso de compra.

  2. Seleccionar Productos:

    3. Se ingresan los códigos de los productos deseados.

  3. Agregar Otro Producto:

    4. Se pregunta si se quiere agregar otro producto.

    Si la respuesta es afirmativa, se repiten los pasos anteriores para agregar más productos.

    Si la respuesta es negativa, se pasa al siguiente paso.

  4. Número de Cuotas al Finalizar:

    5. Se solicita ingresar la cantidad de cuotas deseadas al finalizar la compra.

  5. Finalizar la Compra:

    6. Se calcula y muestra el total del carrito.

    Se calcula y muestra el valor de cada cuota al finalizar.

    Se agradece al usuario por su compra.

    Se limpia la consola.

    Se muestra la información final del carrito, incluyendo el nombre de los productos, el total, el valor de cada cuota y el mensaje de agradecimiento.
